## 📊 기능 및 특징

### ✅ 구현된 기능

1. **환경 감지**
   - OS 타입 (Windows, macOS, Linux, WSL)
   - CPU 아키텍처 (x86_64, ARM64 등)
   - Python 버전 확인

2. **Setup 마법사**
   - 첫 실행 시 자동 실행
   - Gemini API 키 입력
   - 설정 자동 저장

3. **Gemini API 통합**
   - 자연어 → 단계별 시나리오 생성
   - 명령어 실행 결과 설명
   - JSON 기반 구조화된 응답

4. **TUI 레이아웃**
   - 좌측: 세션 목록
   - 중앙: 메인 터미널
   - 우상: 명령어 설명 패널
   - 우하: 에이전트/시나리오 패널

5. **세션 관리**
   - `~/.g-tutor/` 디렉터리 자동 생성
   - 설정 파일 저장 (config.json)
   - 세션 기록 저장 가능

## 🔑 Gemini API 정보

### 무료 티어 한계

| 제한 | 값 |
|------|-----|
| 월별 요청 | 무제한* |
| 일일 요청 | ~250 |
| 분당 요청 | ~10 |
| 사용 모델 | gemini-2.0-flash |

*실제로는 quota 내에서 무제한 가능

### API 특징

- ✅ **무료:** 신용카드 불필요
- ✅ **간단:** 몇 줄의 코드로 시작
- ✅ **강력:** 한국어 지원
- ✅ **신속:** 보통 3~10초 응답

### 보안

- API 키는 `~/.g-tutor/config.json` 에 로컬 저장
- LLM에는 민감 정보 마스킹 후 전송
- 키 유출 시 Google AI Studio에서 즉시 삭제 가능

## 💬 간단한 FAQ

**Q: 정말 무료인가?**
A: 네! Google Gemini API는 무료 티어가 충분합니다.

**Q: 한글만 사용 가능한가?**
A: 아니요. 영문/일문/중문 등 모든 언어 지원.

**Q: 오프라인에서 사용 가능한가?**
A: 아니요. Gemini API 호출이 필요하므로 인터넷 필수.

**Q: 저장된 데이터는 어디에?**
A: 모두 로컬 (`~/.g-tutor/`) 에 저장.
