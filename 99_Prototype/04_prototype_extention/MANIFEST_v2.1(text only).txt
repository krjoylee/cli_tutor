1) gemini_tutor_app_v2.1.py (메인 애플리케이션)
------------------------------------------------
역할: Gemini Tutor CLI의 핵심 프로그램

주요 기능:
  - Textual 기반 대화형 TUI 제공
  - 다중 LLM 통합 (Groq / Perplexity / Gemini)
  - 환경 정보 자동 감지
  - 설정 관리 및 LLM 클라이언트 초기화
  - 명령어 시나리오 생성
  - 명령어 실행 결과 설명

클래스 구조:
  - EnvInfo: OS / CPU 정보 감지
      * _detect_os(): OS 타입 판별
      * _detect_arch(): CPU 아키텍처 감지
      * _check_wsl(): WSL 환경 확인

  - ConfigManager: 설정 파일 관리
      * config_dir: ~/.g-tutor
      * config_file: ~/.g-tutor/config.json
      * save_config(): 설정 저장
      * get() / set(): 설정값 조회/저장

  - SetupWizard: 초기 설정
      * run(): 대화형 설정 진행
      * _setup_groq(): Groq 설정
      * _setup_perplexity(): Perplexity 설정
      * _setup_gemini(): Gemini 설정

  - LLMClient: 다중 LLM 지원
      * generate_scenario(): 명령어 시나리오 생성
      * explain_command(): 명령어 결과 설명
      * _generate_groq(): Groq API 호출
      * _generate_perplexity(): Perplexity API 호출
      * _generate_gemini(): Gemini API 호출

  - UI 패널 (Textual widgets)
      * SessionPanel: 세션 목록 (좌측)
      * TerminalPanel: 터미널 입력/출력 (가운데)
      * ExplanationPanel: 명령어 설명 (우측 상부)
      * AgentPanel: 에이전트 가이드 (우측 하부)

  - GeminiTutorApp: 메인 애플리케이션
      * compose(): UI 레이아웃 구성 (FIXED)
      * action_quit(): 프로그램 종료
      * action_clear(): 화면 지우기

사용 방법:
  python gemini_tutor_app_v2.1.py
    -> 첫 실행 시 Setup 마법사 자동 시작
    -> 이후 TUI 앱 실행

의존성:
  - groq
  - google-generativeai
  - rich
  - textual (권장: 0.40.0 이상)
  - httpx (Perplexity 지원용)

수정 사항 (v2.1):
  - Textual Layout 에서 direction 파라미터 제거
  - Horizontal / Vertical 컨테이너 사용
  - TUI 레이아웃 구조 최적화


2) test_setup_v2.1.py (Setup & 테스트 도구)
------------------------------------------------
역할: 설치 및 환경 검증 도구

주요 기능:
  - 시스템 정보 수집
  - 의존성 확인
  - 저장된 설정 확인
  - API 테스트
  - 시나리오 생성 테스트

클래스/함수:
  - get_processor_info(): CPU 정보
  - get_python_path(): Python 경로
  - load_config(): 설정 파일 로드
  - save_config(): 설정 파일 저장

  - GeminiTutorSetupTool: 메인 클래스
      * run(): 전체 Setup 도구 실행
      * _show_system_info(): 시스템 정보 출력
      * _check_dependencies(): 의존성 확인
      * _check_saved_config(): 설정 확인
      * _test_apis(): API 테스트 메뉴
      * _test_groq(): Groq API 테스트
      * _test_gemini(): Gemini API 테스트
      * _test_perplexity(): Perplexity API 테스트
      * _test_scenario(): 시나리오 생성 테스트

사용 방법:
  python test_setup_v2.1.py
    -> 시스템 정보 표시
    -> 의존성 확인
    -> API 테스트 옵션 제공
    -> 시나리오 생성 테스트

의존성:
  - rich (콘솔 출력)
  - groq (API 테스트)

------------------------------------------------------------

   Linux:    rm ~/.g-tutor/config.json

4. 라이브러리 재설치
   pip install --upgrade groq google-generativeai rich textual httpx
