## 🚀 최초 사용자 5단계 설치

### Step 1: Python 확인 (1분)
```powershell
python --version

```

### Step 2: 라이브러리 설치 (2분)
```powershell

pip install google-generativeai rich textual httpx
```

### Step 3: Setup 마법사 (1분)
```powershell

# 1) 시스템 정보 확인
# 2) 의존성 확인
# 3) LLM 선택 (1=Groq, 2=Perplexity, 3=Gemini)
# 4) API 키 입력
# 5) API 테스트
```

### Step 4: 메인 프로그램 실행 (1분)
```powershell
python gemini_tutor_app_v2.py

# TUI가 나타나고, 명령어 입력 대기 중
```

## 🔑 API 키 발급 방법

### Groq API 키

1. https://console.groq.com 에 접속
2. "Sign Up" (또는 "Sign In") 클릭
3. 이메일/GitHub로 가입/로그인
4. 좌측 "API Keys" 메뉴 클릭
5. "+ Create New API Key" 버튼 클릭
6. 팝업에서 키 이름 입력 (기본값 OK)
7. "Create API Key" 클릭
8. 생성된 키를 복사 (한 번만 표시됨)
9. `gsk_...` 형태의 키를 저장

**제한:**
- 일 1,000 요청
- 분당 30 요청
- 무료

### Google Gemini API 키

1. https://aistudio.google.com 에 접속
2. Google 계정으로 로그인
3. "Get API Key" 버튼 (좌측 상단)
4. "Create API key in new project" 선택
5. "Create API key" 클릭
6. 생성된 키 복사
7. `AIzaSyD_...` 형태의 키를 저장

**제한:**
- 일 250 요청
- 무료
- 신용카드 불필요

### Perplexity API 키

1. https://www.perplexity.ai 에 접속
2. 계정 생성/로그인
3. 우측 상단 프로필 → "Settings" 클릭
4. "API" 탭 선택
5. "Create New API Key" 버튼
6. 키 이름 입력 후 생성
7. `ppl_...` 형태의 키 복사 및 저장

**제한:**
- 요금제에 따라 상이
- 무료 시험 (첫 5달러)

## 🐛 공통 문제 해결

### 1. "No module named 'groq'"
```powershell
pip install groq google-generativeai rich textual httpx
```

### 2. "API 키 오류"
- 설정 파일 삭제: `~/.g-tutor/config.json`
- Setup 다시 실행
- API 키 다시 입력

### 3. "인터넷 연결 실패"
```powershell
ping google.com
# 또는 각 API 서버 ping
```

### 4. "권한 오류"
```powershell
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
```
