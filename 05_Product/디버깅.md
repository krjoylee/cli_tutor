## 🐞 발견된 이슈 및 해결 이력

### [Bug #1] TCSS 스타일시트 파싱 에러
- **발생 시각**: 2026-03-10 19:13
- **증상**: `python -m cli_tutor` 실행 시 `Expected selector or end of file (found ':focus {\n').` 에러와 함께 앱 실행 실패.
- **원인**: `app.tcss` 내에서 의사 클래스 `:focus`를 단독 선택자로 사용함. Textual CSS는 표준 CSS와 달리 의사 클래스 단독 사용을 허용하지 않음.
- **해결**: `:focus`를 `*:focus`로 변경하여 모든 요소의 포커스 상태를 대상으로 하도록 수정.
- **결과**: ✅ 해결 완료 (260310 19:15)

### [Bug #5] Windows Subprocess 파이프 충돌 (Panic)
- **발생 시각**: 2026-03-10 19:47
- **증상**: 앱 실행 중 또는 종료 시 `ValueError: I/O operation on closed pipe` 발생하며 튕김.
- **원인**: Python 3.11+ 윈도우 환경에서 `asyncio`의 `ProactorEventLoop`가 서브프로세스 종료 시 파이프를 닫는 순서에서 발생하는 고질적인 이슈.
- **해결**: 서브프로세스 자원 해제 루틴 강화 및 예외 트래핑 추가. (진행 중)

## 🛠️ CLI Tutor 디버깅 방법론 (Debugging Methodology)

초보자도 쉽게 원인을 파악하고 전달할 수 있도록 아래 원칙에 따라 개발/디버깅을 진행합니다.

1.  **Fail-Safe Logging (생존 로그)**:
    *   앱이 튕기더라도 터미널 화면에 "어디서 튕겼는지" 마지막 유언을 남기도록 코드를 작성합니다. (`print` 문 적극 활용)
2.  **UI Feedback Transparency (투명한 피드백)**:
    *   성공 시: 패널 테두리를 녹색으로 변경하고 `[성공]` 메시지 출력.
    *   실패 시: 패널 테두리를 빨간색으로 변경하고 `[에러사유]`를 패널 내부에 직접 렌더링.
3.  **Step-by-Step Verification (단계별 검증)**:
    *   `/goal` 입력 → `워커 기동` → `LLM 호출` → `응답 수신` → `파싱` → `출력` 각 단계마다 내부 로그를 남겨 병목 지점을 찾습니다.
4.  **Issue Reproduction (재현 경로 기록)**:
    *   문제 발생 시 입력한 명령어와 환경(터미널 종류 등)을 반드시 `디버깅.md`에 기록합니다.

## 🛠️ CLI Tutor 디버깅 방법론 (Debugging Methodology) - v1.2 보완

7.  **Verbose Post-Mortem (사후 원인 보고)**:
    *   사용자가 앱을 강제 종료(`Ctrl+C`)하거나 앱이 튕기더라도, 터미널(stdout)에 마지막으로 수행된 작업의 성공/실패 여부를 사람이 읽기 쉬운 메시지로 남깁니다.
    *   수행 내역: `[DEBUG] [SUCCESS] 시나리오 가이드 로드 완료` 등의 메시지 출력.
8.  **Atomic UI Reset (원자적 화면 초기화)**:
    *   명령어 제출 즉시 비동기 작업 시작 전 화면을 비우는 것을 원칙으로 하여 "멈춰있는 느낌"을 제거합니다.

9.  **Persistent Log Files (영구 로그 기록)**:
    *   앱 실행 시마다 `logs/cli_tutor_YYYYMMDD.log` 파일에 배후의 모든 동작(Subprocess 시작, LLM 요청/응답 등)을 기록합니다.
    *   앱이 튕기거나 화면이 멈췄을 때, 이 로그 파일을 확인하면 사용자가 무엇을 했고 어디서 막혔는지 100% 추적 가능합니다.

10. **Third-party Log Filtering (로그 노이즈 제거)**:
    *   `markdown_it`, `httpx` 등 외부 라이브러리에서 발생하는 불필요한 `[DEBUG]` 메시지(`StateBlock`, `HTTP Request` 상세 등)를 차단하여 실무적인 디버깅 정보만 남깁니다.
