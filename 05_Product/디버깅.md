## 🐞 발견된 이슈 및 해결 이력

### [Bug #1] TCSS 스타일시트 파싱 에러
- **발생 시각**: 2026-03-10 19:13
- **증상**: `python -m cli_tutor` 실행 시 `Expected selector or end of file (found ':focus {\n').` 에러와 함께 앱 실행 실패.
- **원인**: `app.tcss` 내에서 의사 클래스 `:focus`를 단독 선택자로 사용함. Textual CSS는 표준 CSS와 달리 의사 클래스 단독 사용을 허용하지 않음.
- **해결**: `:focus`를 `*:focus`로 변경하여 모든 요소의 포커스 상태를 대상으로 하도록 수정.
- **결과**: ✅ 해결 완료 (260310 19:15)

### [Bug #2] border 속성 색상값 오류
- **발생 시각**: 2026-03-10 19:20
- **증상**: `Invalid value for border property` 에러 발생. `tall bright_cyan` 및 `tall bright_green`이 유효하지 않은 값으로 처리됨.
- **원인**: TCSS의 `border` 단축 속성에서 `bright_` 계열의 색상 명칭 파싱 문제로 추정.
- **결과**: ✅ 해결 완료 (260310 19:30)
